Si tienen instalada alguna versión de python que no sea la 3.7 desinstalenla

Si no tienen cuenta en Github, regístrense acá:
https://github.com/join

Pasen su usuario de Github así los agrego al repositorio

Descarguen Python de acá:
https://www.python.org/ftp/python/3.6.6/python-3.6.6-amd64.exe

Descarguen Pycharm de acá:		
https://www.jetbrains.com/pycharm/download/download-thanks.html?platform=windows&code=PCC

Descarguen Github de acá:
https://central.github.com/deployments/desktop/desktop/latest/win32

En el instalador de Python:
- Marquen "Add Python 3.6 to PATH"
- Customize Installation
- Siguiente
- Marquen Install for all users, si quieren cambien la carpeta e instalen

Cuando instalen Pycharm (instalen primero python), marquen ".py".
Para instalar Github, nada distinto, todo siguiente nomás.

Abran "Ejecutar"->cmd->pip3 install --user PyQt5
pip3 install --user PyQt5-tools

Abran github, pongan "Clone a respository" y les debería aparecer uno que dice SO, si quieren cambien la carpeta y clonen.
Cuando hagan cambios en los archivos de esa carpeta, en github les van a aparecer los cambios; abajo a la izquierda ponen un título y si tienen ganas una descripción de lo que hicieron y hacen el commit.
